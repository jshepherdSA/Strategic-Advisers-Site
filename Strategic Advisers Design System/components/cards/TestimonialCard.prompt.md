**TestimonialCard** — a client quote with attribution. Large red quote mark, display-font quote and a rule-led attribution row. Use `tone="navy"` for a feature panel.

```jsx
<TestimonialCard
  quote="They turned a difficult announcement into a moment of trust."
  name="Dana Whitfield"
  role="VP Communications"
  org="Meridian Energy"
  tone="navy"
/>
```
