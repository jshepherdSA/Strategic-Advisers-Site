**Badge** — small uppercase mono pill for categories, tags and statuses.

```jsx
<Badge variant="accent">Public Affairs</Badge>
<Badge variant="navy">Case Study</Badge>
<Badge variant="on-navy">Featured</Badge>
```

Variants: `neutral`, `navy`, `accent` (soft red), `solid` (navy), `solid-accent` (red), `success`, `on-navy` (for dark sections). Sizes `sm` `md`.
