**SectionHeader** — the standard section opener: eyebrow, large display headline, optional intro. Left-aligned by default (pass an `action` Button to sit on the right), or centered.

```jsx
<SectionHeader
  eyebrow="What we do"
  title="Strategy that moves the conversation"
  intro="Integrated communications across PR, digital media and public affairs."
  action={<Button variant="outline">All services</Button>}
/>
```

Use `tone="on-navy"` inside dark sections.
