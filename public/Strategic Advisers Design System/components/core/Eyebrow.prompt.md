**Eyebrow** — the brand's signature mono kicker that sits above a headline. Wide letter-spacing, uppercase, red by default, with a short leading rule.

```jsx
<Eyebrow>What we do</Eyebrow>
<Eyebrow color="on-navy">Public Affairs</Eyebrow>
<Eyebrow rule={false} color="muted">03 / 06</Eyebrow>
```

Props: `rule` (leading dash, default true), `color` (`accent` | `navy` | `muted` | `on-navy`).
